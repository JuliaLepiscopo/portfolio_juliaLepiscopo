# Portfólio — Julia Lepiscopo

Portfólio pessoal desenvolvido em **React** + **Tailwind CSS**.

## ✨ Sobre o projeto

Site de portfólio organizado em sessões, construído de forma incremental.
O acompanhamento do desenvolvimento é feito por **milestones**, **issues** e **sub-issues** aqui no GitHub.

## 🧩 Sessões do portfólio

| # | Sessão | Descrição |
|---|--------|-----------|
| 1 | 👋 Boas-vindas | Tela de entrada / apresentação inicial |
| 2 | 🚀 Início | Hero principal com chamada e navegação |
| 3 | 🙋 Sobre mim | Apresentação pessoal e trajetória |
| 4 | 🎓 Graduações | Formação acadêmica |
| 5 | 🛠️ Skills | Competências e tecnologias |
| 6 | 💼 Experiência | Experiência profissional |
| 7 | 📂 Projetos | Projetos desenvolvidos |
| 8 | 🏁 Finalização | Contato e encerramento |

## 🛠️ Stack

- [React](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)

## 🚀 Como rodar (em breve)

> O scaffold do projeto será criado na milestone de **Setup & Infraestrutura**.

```bash
npm install
npm run dev
```
